## 📌 Project Overview

Employee attrition is one of the most important HR metrics, directly affecting recruitment costs, workforce productivity, and organizational growth. This project analyzes employee data to identify workforce trends, measure attrition, and generate actionable business insights.

The project has been implemented using **SQL**, **Microsoft Excel**, and **Power BI**, demonstrating the complete analytics workflow from data exploration to interactive dashboard development.

---

## 🎯 Business Problem

The HR department requires a centralized analytics solution to monitor employee performance indicators and workforce trends. The existing process lacks clear KPIs and visual reports to track employee attrition and organizational demographics.

This dashboard helps HR teams:

- Monitor employee count
- Track employee attrition
- Calculate attrition rate
- Analyze workforce demographics
- Identify departments with high attrition
- Measure employee satisfaction
- Support data-driven HR decisions
